Konverti can currently convert files to the following formats:

WAV (uncompressed: 8-bit, 16-bit, 24-bit, 32-bit, 32-bit float, 64-bit float)
AIFF (uncompressed: 8-bit, 16-bit, 16-bit OS X (sowt), 24-bit, 32-bit)
MP3 (VBR and CBR)

*** STILL IN PROGRESS ***
FLAC {not implemented yet}
OGG Vorbis (libvorbis or vorbis codecs) {need to add for custom use of flac, incl. disable quality combo box and add new check for enabling selection from main window}
M4A {Preset only for now; need to add custom, including using multiple codecs}
